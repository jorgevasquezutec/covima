## Resumen Ejecutivo

### Objetivo
Crear un sistema de gamificacion para incentivar la participacion de los jovenes en las actividades de JA mediante:
- Rankings competitivos por trimestre
- Sistema de puntos por asistencias y participaciones
- Niveles biblicos con nombres creativos
- Insignias y logros desbloqueables
- Premios por participacion y nivel

### Funcionalidades Principales
1. **Rankings trimestrales** - Competencia sana entre participantes
2. **Puntos diferenciados** - Cada tipo de participacion tiene un valor distinto
3. **Confirmacion de participacion** - Admin confirma quien realmente participo
4. **Niveles biblicos** - Progresion con nombres tematicos
5. **Reavivados por su Palabra** - Registro de ganadores para el ranking
6. **Escuela Sabatica** - Registro de quien dirige

## Rankings por Trimestre

### Definicion de Trimestres

| Trimestre | Meses | Fechas |
|-----------|-------|--------|
| Q1 | Enero - Marzo | 01/01 - 31/03 |
| Q2 | Abril - Junio | 01/04 - 30/06 |
| Q3 | Julio - Septiembre | 01/07 - 30/09 |
| Q4 | Octubre - Diciembre | 01/10 - 31/12 |

### Tipos de Rankings

1. **Ranking General** - Suma de todos los puntos del trimestre
2. **Ranking de Asistencia** - Solo puntos de asistencia
3. **Ranking de Participacion** - Solo puntos de participacion en programas
4. **Ranking de Racha** - Mayor racha del trimestre

### Query de Ejemplo

```sql
-- Ranking general del trimestre
SELECT
  u.id,
  u.nombre,
  ug.nivel,
  ug.nivelNombre,
  SUM(hp.puntos) as puntosTrimestre
FROM Usuario u
JOIN UsuarioGamificacion ug ON u.id = ug.usuarioId
JOIN HistorialPuntos hp ON ug.id = hp.usuarioGamId
WHERE hp.trimestre = 1 AND hp.anio = 2026
GROUP BY u.id, u.nombre, ug.nivel, ug.nivelNombre
ORDER BY puntosTrimestre DESC
LIMIT 10;
```

```typescript
// backend/src/gamificacion/gamificacion.service.ts

@Injectable()
export class GamificacionService {
  constructor(private prisma: PrismaService) {}

  async confirmarParticipacionYAsignarPuntos(
    asistenciaId: number,
    adminId: number,
    tipoParticipacion: string // codigo del ConfiguracionPuntaje
  ) {
    return this.prisma.$transaction(async (tx) => {
      // 1. Confirmar asistencia
      const asistencia = await tx.asistencia.update({
        where: { id: asistenciaId },
        data: {
          estado: 'confirmado',
          confirmadoPor: adminId,
          confirmadoAt: new Date(),
        },
        include: { usuario: true },
      });

      // 2. Obtener configuracion de puntaje
      const configPuntaje = await tx.configuracionPuntaje.findUnique({
        where: { codigo: tipoParticipacion },
      });

      if (!configPuntaje || !configPuntaje.activo) {
        throw new Error('Tipo de participacion no valido');
      }

      // 3. Obtener o crear gamificacion del usuario
      let usuarioGam = await tx.usuarioGamificacion.findUnique({
        where: { usuarioId: asistencia.usuarioId },
      });

      if (!usuarioGam) {
        usuarioGam = await tx.usuarioGamificacion.create({
          data: { usuarioId: asistencia.usuarioId },
        });
      }

      // 4. Calcular trimestre actual
      const fecha = new Date();
      const trimestre = Math.ceil((fecha.getMonth() + 1) / 3);
      const anio = fecha.getFullYear();

      // 5. Registrar puntos en historial
      await tx.historialPuntos.create({
        data: {
          usuarioGamId: usuarioGam.id,
          puntos: configPuntaje.puntos,
          tipo: configPuntaje.categoria,
          descripcion: `${configPuntaje.nombre} - Programa`,
          referenciaId: asistenciaId,
          referenciaTipo: 'asistencia',
          trimestre,
          anio,
        },
      });

      // 6. Actualizar totales
      const nuevosPuntos = usuarioGam.puntosTotal + configPuntaje.puntos;
      const nuevaXP = usuarioGam.experiencia + configPuntaje.puntos;

      // 7. Verificar nuevo nivel
      const nuevoNivel = await this.calcularNivel(tx, nuevaXP);

      // 8. Actualizar usuario
      await tx.usuarioGamificacion.update({
        where: { id: usuarioGam.id },
        data: {
          puntosTotal: nuevosPuntos,
          experiencia: nuevaXP,
          nivel: nuevoNivel.nivel,
          nivelNombre: nuevoNivel.nombre,
          ultimaActividad: new Date(),
        },
      });

      // 9. Verificar insignias desbloqueables
      await this.verificarInsignias(tx, usuarioGam.id);

      return {
        puntosAsignados: configPuntaje.puntos,
        nuevoTotal: nuevosPuntos,
        nivelActual: nuevoNivel,
      };
    });
  }

  private async calcularNivel(tx: any, experiencia: number) {
    const nivel = await tx.nivelBiblico.findFirst({
      where: { experienciaMin: { lte: experiencia } },
      orderBy: { experienciaMin: 'desc' },
    });
    return nivel || { nivel: 1, nombre: 'Discipulo' };
  }

  private async verificarInsignias(tx: any, usuarioGamId: number) {
    // TODO: Implementar logica de verificacion de insignias
  }

  async getRanking(trimestre: number, anio: number, tipo: string = 'general') {
    const whereClause: any = { trimestre, anio };

    if (tipo !== 'general') {
      whereClause.tipo = tipo;
    }

    return this.prisma.$queryRaw`
      SELECT
        u.id,
        u.nombre,
        u."fotoUrl",
        ug.nivel,
        ug."nivelNombre",
        COALESCE(SUM(hp.puntos), 0)::int as "puntosTrimestre"
      FROM "Usuario" u
      LEFT JOIN "UsuarioGamificacion" ug ON u.id = ug."usuarioId"
      LEFT JOIN "HistorialPuntos" hp ON ug.id = hp."usuarioGamId"
        AND hp.trimestre = ${trimestre}
        AND hp.anio = ${anio}
        ${tipo !== 'general' ? Prisma.sql`AND hp.tipo = ${tipo}` : Prisma.empty}
      WHERE u.activo = true
      GROUP BY u.id, u.nombre, u."fotoUrl", ug.nivel, ug."nivelNombre"
      ORDER BY "puntosTrimestre" DESC
      LIMIT 50
    `;
  }

  async getMiProgreso(usuarioId: number) {
    const gamificacion = await this.prisma.usuarioGamificacion.findUnique({
      where: { usuarioId },
      include: {
        historialPuntos: {
          orderBy: { createdAt: 'desc' },
          take: 20,
        },
        insignias: {
          include: { insignia: true },
        },
      },
    });

    // Obtener siguiente nivel
    const siguienteNivel = await this.prisma.nivelBiblico.findFirst({
      where: { experienciaMin: { gt: gamificacion?.experiencia || 0 } },
      orderBy: { experienciaMin: 'asc' },
    });

    // Calcular posicion en ranking actual
    const fecha = new Date();
    const trimestre = Math.ceil((fecha.getMonth() + 1) / 3);
    const anio = fecha.getFullYear();

    const posicion = await this.calcularPosicionRanking(usuarioId, trimestre, anio);

    return {
      ...gamificacion,
      siguienteNivel,
      xpParaSiguienteNivel: siguienteNivel
        ? siguienteNivel.experienciaMin - (gamificacion?.experiencia || 0)
        : null,
      posicionRanking: posicion,
      trimestre,
      anio,
    };
  }
```
